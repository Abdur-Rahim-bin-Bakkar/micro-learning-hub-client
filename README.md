# 🌍 Introduction

Education is becoming more digital every day, but many online learning platforms still lack a complete ecosystem where students, teachers, and administrators can collaborate seamlessly.

**Micro Learning Hub** was created to solve this problem.

It is a modern, scalable, and feature-rich Learning Management System (LMS) that enables students to learn online, teachers to share knowledge, and administrators to manage the entire platform efficiently.

Unlike traditional LMS platforms, Micro Learning Hub is not only focused on course management—it also provides online examinations, teacher applications, community discussions, announcements, secure authentication, course enrollment, and much more.

The platform is designed with a clean UI, responsive layouts, modern technologies, and scalable architecture to provide the best learning experience.

---

# 📖 About The Project

Micro Learning Hub is a Full-Stack MERN-based Learning Management System built to simplify online education for everyone.

The platform allows students to explore available courses, purchase and enroll in premium courses, participate in online MCQ examinations, monitor their learning progress, and engage with teachers and classmates through the integrated Help Desk community.

Teachers can apply to become instructors, create educational courses, publish online examinations, manage questions, monitor student performance, and interact with learners through educational discussions.

Administrators have full control over the system. They can approve teacher applications, manage users, publish announcements, monitor activities, and ensure the platform operates smoothly.

This project combines multiple educational tools into one modern web application, creating a complete digital learning environment.

# ⚙️ Installation Guide

Follow these steps to run **Micro Learning Hub** on your local machine.

---

## 📌 Prerequisites

Before installing the project, make sure you have installed:

* Node.js
* npm or yarn
* MongoDB
* Git

---

# 📥 Clone The Repository

```bash
git clone https://github.com/your-username/micro-learning-hub.git
```

Go to the project directory:

```bash
cd micro-learning-hub
```

---

# 💻 Frontend Setup

Navigate to the client folder:

```bash
cd client
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=your_backend_api_url
NEXT_PUBLIC_IMGBB_KEY=your_imgbb_api_key
```

Run the frontend:

```bash
npm run dev
```

Frontend will run on:

```
http://localhost:3000
```

---

# 🖥️ Backend Setup

Navigate to the server folder:

```bash
cd server
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
PORT=5000

DATABASE_URL=your_mongodb_connection_string

JWT_SECRET=your_secret_key

IMGBB_API_KEY=your_imgbb_api_key
```

Run backend server:

```bash
npm run dev
```

Backend will run on:

```
http://localhost:5000
```
